資料蒐集

本應用程式（Liam AI）僅用於管理 From Source To TABLE 粉絲專頁的內容發布。本應用程式不蒐集、儲存或分享任何用戶個人資料。

Facebook 資料使用

本應用程式透過 Meta Graph API 存取粉絲專頁，僅用於自動發布貼文內容。所取得的存取權限（pages_manage_posts、pages_read_engagement）僅供粉絲專頁管理之用，不作其他用途。

資料儲存

本應用程式不儲存任何來自 Facebook 的用戶資料或貼文互動資料。

第三方共享

本應用程式不與任何第三方共享資料。
